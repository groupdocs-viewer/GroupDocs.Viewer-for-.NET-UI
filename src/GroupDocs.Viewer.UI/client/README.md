# Angular UI for GorupDocs.Viewer.UI

Building this app requires legacy tools that can be installed with `nvm`.

### Prepare

```bash
nvm install 14.15  
nvm use 14.15.5  
npm install -g @angular/cli@14.2.13  
npm install  
```

### Start dev server 

```bash
ng serve
```

### Build release packages

```bash
npm run build
```
